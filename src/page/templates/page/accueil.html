Le projet edifi6

Un projet en quatre dimensions

Modélisation

Elaboration d'un modèle qui rende les données Interopérables, Pérennes et Accessibles.
Application

Application du modèle, construite sur mesure et open source.
Documentation

Le projet est documenté : définitions, schémas, spécifications, etc.
Communication

Les données au coeur de l'indispensable transition climatique.
Modélisation

Un modèle de données original

Ce modèle est basé sur des normes législatives et industrielles, sur des sources authentiques et, surtout, sur les principes du BIM.

Application

edifi6, un outil modulable

L'application - encore en phase de prototype - est une mise en oeuvre de notre Modèle dans la comptabilité énergétique des bâtiments.

Sur mesure

L'outil s'adapte, et non pas l'utilisateur : edifi6 est écrit sur mesure, en fonction des besoins et réalités.

Open source

Le logiciel est open source, il bénéficie donc des avantages qui y sont liés (fiabilité, sécurité, disponibilité,...).

Les définitions utilisées sont reprises dans le glossaire. Le code, les schémas et autres documents relatifs sont disponibles sur Github.

Plaidoyer

Convaincre. Encore. Toujours.

Les enjeux sont énormes, les techniques relativement récentes et les résistances nombreuses : il faut, sans relâche, convaincre que les changements sont nécessaires et, surtout, qu'ils sont accessibles.
